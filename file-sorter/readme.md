# File Sorter

A Python automation script that organizes files in a directory into folders
based on file type.

The script automatically creates folders and moves files without overwriting
existing ones.

## Features
- Sorts files into categories (Images, Documents, Spreadsheets, etc.)
- Automatically creates folders if missing
- Prevents file overwriting
- Useful for basic directory cleanup

## Concepts Used
- File handling
- os module
- shutil module
- Conditional logic
