# Rock Paper Scissors Game

A console-based Rock-Paper-Scissors game where the user plays against
the computer.

The game keeps track of scores for both the user and the computer.

## Features
- User vs computer gameplay
- Random computer choices
- Score tracking
- Replay-friendly loop

## Concepts Used
- Conditional logic
- Loops
- random module
- String comparison

## How to Run
```bash
python rock_paper_scissors.py
