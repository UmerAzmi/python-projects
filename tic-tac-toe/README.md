# Tic Tac Toe

A console-based Tic Tac Toe game built using Python where the user
plays against the computer.

The computer uses basic decision-making to try to win or block the
player before choosing a random move.

## Features
- User vs computer gameplay
- 3Ã—3 board displayed in the console
- Input validation for invalid or occupied positions
- Win and draw detection
- Computer move logic (win, block, then random)

## Game Rules
- You play as `X`
- The computer plays as `O`
- Players take turns placing their symbol
- First to get three in a row wins
- If the board fills with no winner, the game ends in a draw

## Concepts Used
- Lists
- Functions
- Conditional statements
- Loops
- Random module
- Game state management
