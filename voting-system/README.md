# Voting System

A simple console-based Voting System built using Python.
The application allows users to cast votes for predefined candidates
and view the current voting results.

This project was created to practice working with dictionaries,
user input validation, and menu-driven programs.

## Features
- Cast votes for candidates (A, B, or C)
- Prevents invalid voting options
- View real-time voting results
- Menu-driven interface
- Continuous execution until exit

## Concepts Used
- Dictionaries
- Functions
- while loops
- Conditional statements
- String methods (`upper()`)

## How to Run
```bash
python voting_system.py
