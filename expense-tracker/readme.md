# Expense Tracker

A simple console-based Expense Tracker application built using Python.
The program allows users to record expenses, view all entries, and
calculate the total expense for a session.

This project was created to practice handling numeric input, lists,
tuples, and basic calculations in Python.

## Features
- Add expenses with name and amount
- View all recorded expenses
- Calculate total expense
- Menu-driven interface
- Basic error handling for invalid input

## Concepts Used
- Lists and tuples
- Functions
- while loops
- Conditional statements
- sum() function
- try / except for input validation

## How to Run
```bash
python expense_tracker.py
