# Quiz Game (Python)

A collection of console-based quiz games built using Python to practice
basic programming concepts and logic.

This folder contains two versions of the quiz game, showing progression
from a simple implementation to a more structured one.

## Projects Included

### Quiz Game
- Simple multiple-choice quiz
- Uses basic input/output and conditional statements
- Calculates score and percentage

### Quiz Game 2
- Improved version using dictionaries
- Stores questions and answers in structured data
- Cleaner and more scalable logic

## Concepts Used
- Variables
- Conditionals
- Loops
- Dictionaries
- User input handling

## How to Run
```bash
python quiz_game.py
python quiz_game2.py
