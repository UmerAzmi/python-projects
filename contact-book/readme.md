# Contact Book

A simple console-based Contact Book application built using Python.
The program allows users to store, view, and search contacts during
a single session.

This project was created to practice working with dictionaries,
functions, loops, and user input handling.

## Features
- Add new contacts with name and phone number
- View all saved contacts
- Search for a contact by name (case-insensitive)
- Menu-driven interface
- Runs continuously until the user exits

## Concepts Used
- Dictionaries
- Functions
- while loops
- Conditional statements
- User input and output
- Basic string manipulation

## How to Run
```bash
python contact_book.py
